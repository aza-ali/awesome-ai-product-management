## Tools for PMs

AI-native or AI-augmented tools PMs actually use day to day.

### Research & synthesis

- [Claude](https://claude.ai) — Long-context LLM for spec writing, research, and doc analysis.
- [NotebookLM](https://notebooklm.google) — Multi-document Q&A across an uploaded corpus.
- [Perplexity](https://perplexity.ai) — Cited-source AI search for market and competitive research.

### Prototyping & code

- [Claude Code](https://code.claude.com/docs/en/overview) — Terminal agent for code-level PM artifacts.
- [Cursor](https://cursor.com) — AI-first IDE for prototyping and reading codebases.
- [v0](https://v0.dev) — Generates UI and full apps from text prompts.

### Docs & tracking

- [Coda](https://coda.io) — Formula-driven AI docs for living strategy.
- [Linear](https://linear.app) — Project tracker with first-class AI agent triage.
- [Notion AI](https://www.notion.com/product/ai) — AI-augmented docs and databases for PRDs.
- [Productboard](https://www.productboard.com) — Product platform with AI feedback synthesis.

### User feedback

- [Dovetail](https://dovetail.com) — AI-tagged user research repository with theme extraction.
- [Granola](https://www.granola.ai) — AI meeting capture and summarization.
- [Kraftful](https://kraftful.com) — Synthesizes app reviews, tickets, and surveys into themes.

## Contributing

Open a PR to add a resource. Guidelines:

- One resource per PR.
- Place additions alphabetically within the relevant section.
- Format: `[Name](URL) — Short description ending with a period.`
- Description should be one sentence, descriptive, never marketing.
- Resource must be substantive — no thin link-collections, no paid-only content without a significant free portion, no general productivity tools without a clear PM/AI angle.

If you're unsure whether something fits, open an issue first.
