## Tools for PMs

AI-native or AI-augmented tools PMs actually use day to day.

### Research & synthesis

- <a href="https://claude.ai"><img src="./text/claude.svg" align="absmiddle" alt="Claude" /></a> — Long-context LLM for spec writing, research, and doc analysis.
- <a href="https://notebooklm.google"><img src="./text/notebooklm.svg" align="absmiddle" alt="NotebookLM" /></a> — Multi-document Q&A across an uploaded corpus.
- <a href="https://perplexity.ai"><img src="./text/perplexity.svg" align="absmiddle" alt="Perplexity" /></a> — Cited-source AI search for market and competitive research.

### Prototyping & code

- <a href="https://code.claude.com/docs/en/overview"><img src="./text/claude-code.svg" align="absmiddle" alt="Claude Code" /></a> — Terminal agent for code-level PM artifacts.
- <a href="https://cursor.com"><img src="./text/cursor.svg" align="absmiddle" alt="Cursor" /></a> — AI-first IDE for prototyping and reading codebases.
- <a href="https://v0.dev"><img src="./text/v0.svg" align="absmiddle" alt="v0" /></a> — Generates UI and full apps from text prompts.

### Docs & tracking

- <a href="https://coda.io"><img src="./text/coda.svg" align="absmiddle" alt="Coda" /></a> — Formula-driven AI docs for living strategy.
- <a href="https://linear.app"><img src="./text/linear.svg" align="absmiddle" alt="Linear" /></a> — Project tracker with first-class AI agent triage.
- <a href="https://www.notion.com/product/ai"><img src="./text/notion-ai.svg" align="absmiddle" alt="Notion AI" /></a> — AI-augmented docs and databases for PRDs.
- <a href="https://www.productboard.com"><img src="./text/productboard.svg" align="absmiddle" alt="Productboard" /></a> — Product platform with AI feedback synthesis.

### User feedback

- <a href="https://dovetail.com"><img src="./text/dovetail.svg" align="absmiddle" alt="Dovetail" /></a> — AI-tagged user research repository with theme extraction.
- <a href="https://www.granola.ai"><img src="./text/granola.svg" align="absmiddle" alt="Granola" /></a> — AI meeting capture and summarization.
- <a href="https://kraftful.com"><img src="./text/kraftful.svg" align="absmiddle" alt="Kraftful" /></a> — Synthesizes app reviews, tickets, and surveys into themes.

## Contributing

Open a PR to add a resource. Guidelines:

- One resource per PR.
- Place additions alphabetically within the relevant section.
- Format: `[Name](URL) — Short description ending with a period.`
- Description should be one sentence, descriptive, never marketing.
- Resource must be substantive — no thin link-collections, no paid-only content without a significant free portion, no general productivity tools without a clear PM/AI angle.

If you're unsure whether something fits, open an issue first.
